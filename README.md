# Dorolien — Корпоративная веб-платформа

CRM + Складской учет + Бухгалтерия + Аналитика медицинских представителей

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Создание базы данных
```bash
python manage.py migrate
```

### 3. Загрузка тестовых данных
```bash
python manage.py seed_data
```

### 4. Запуск сервера
```bash
python manage.py runserver
```

Откройте браузер: **http://localhost:8000**

## 📋 Модули системы

- **Дашборд** — ключевые показатели, предупреждения, топы
- **CRM** — врачи, аптеки, визиты с GPS/фото/аудио
- **Склад** — товары, партии, сроки годности, движения
- **Продажи** — оформление, маржа, аналитика
- **Отчеты** — по сотрудникам, аптекам, препаратам

## ⚙️ Стек

- **Backend:** Django 4.2
- **Database:** SQLite (легко переключить на PostgreSQL)
- **Templates:** Django Templates
- **Auth:** Django Auth + кастомная модель User с ролями
